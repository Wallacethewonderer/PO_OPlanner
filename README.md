# PO_OPlanner
A simple daily planner app

## How to Use
You can type your to-dos in the timeblocks and then click the blue buttons to save.

## Technologies
Based on HTML, CSS, JavaScript, JQuery, Dayjs, Bootstrap, Google Fonts.
